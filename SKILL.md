# IR文献综述写作 Skill

## 核心写作哲学

**文献不是单位，问题和张力才是单位。**

综述的任务不是"报告别人说了什么"，而是"用已有研究呈现一个结构性张力，并在这个张力中确立本研究的位置"。好的IR文献综述读起来像一个论证，而不是一份目录。

## 文献评价的核心目的：正当性与合理性

文献评价最基本目的是同时赋予研究尝试**正当性**与**合理性**（王凯、唐世平）：

- **正当性（legitimacy）**：本研究是否值得做？既有解释存在什么问题，使得新研究有必要？
- **合理性（feasibility）**：本研究是否可以执行？从概念、理论、方法、资料上是否可行？

好的文献评价接近一个简略版的研究设计——既告诉读者"既有研究在具体X点上存在具体问题"，也告诉读者"本研究通过Y来解决"。

## 第零步：确认输入材料

在开始任何分析之前，先确认用户提供了哪些材料：

| 确认项 | 说明 |
|---|---|
| 论文主题与研究问题 | 研究对象、核心概念、研究类型（描述性/解释性） |
| 已有文献材料 | 论文框架、参考文献列表、摘要、PDF、阅读笔记 |
| 论文框架（可选） | 若已成型可提供，用于后续研究定位 |
| 目标输出 | 文献底稿 / 综述草稿 / 直接成稿 docx |

如果用户只给了题目而没有文献，先请用户补充文献材料。不做文献检索。

---

## 第一步：建立文献地图与功能分层

将用户提供的文献按功能分类。先做证据表，再分类。

**证据表**（每篇文献一条记录，内部维护即可）：

| 字段 | 说明 |
|---|---|
| 作者-年份 | 引用标识 |
| 标题 | 完整标题 |
| 核心论断 | 一句话：这篇文章最终主张什么？ |
| 方法与材料 | 研究设计、案例/数据、方法类型 |
| 研究对象与范围 | 时间、地区、分析层次 |
| 对当前研究的功能 | 竞争 / 概念支撑 / 方法支撑 / 背景补充 / 反例修正 |
| 使用状态 | 已确认可使用 / 待补充信息 |

**文献功能分层**（必须明确归类）：

- **竞争性文献（内圈，批评的首要靶子）**：已尝试回答同一研究问题的文献。这是综述的主体，需要详写——从四维评价中选取最有力的批评角度，集中论证。
- **概念性文献**：界定核心概念。功能是"证明本研究使用的概念有据可循"，篇幅宜简。
- **方法性文献**：支撑研究设计。功能是"证明本研究的路径可以执行"，不宜展开批评。
- **背景性文献（外圈）**：提供领域脉络。一笔带过或合并处理。

篇幅分配原则：最难处理的竞争性文献给最多篇幅；背景性文献合并或一笔带过。

---

## 第一点五步：核心文献精读——四维评价框架

对竞争性文献逐一精读，按以下四维评价（Knopf 2006）：

1. **假设（Assumptions）**：核心前提是否成立？去掉这个假设，论断还成立吗？争论可否追溯到假设不同？
2. **逻辑（Logic）**：因果推理是否自洽？有无内在矛盾或关键跳跃？最有力的反论是什么？作者是否充分回应？
3. **证据（Evidence）**：证据是否有效、有代表性？是否存在选择性偏误？是否考虑了所有明显相关的案例或数据？
4. **方法（Methodology）**：研究设计是否适合回答该问题？方法分歧是否构成争论的底层基础？

精读时优先从作者自己承认的局限性段落中提取批评线索；其次从四维评价中识别"A的解释在B条件下失效，因为……"的张力。

---

## 第二步：识别核心张力与缺口三分法

先判断研究类型，再定位核心张力：

| 研究特征 | 核心张力通常在于 |
|---|---|
| 提出新解释变量 | 既有文献忽视了X对Y的影响 |
| 修正既有理论 | 既有解释在机制/条件/范围上存在缺陷 |
| 新概念/重新界定概念 | 既有概念模糊或错误划分了现象边界 |
| 新案例/新数据 | 既有结论的外部有效性存疑 |
| 跨层次/跨子领域 | 两个研究传统之间存在对话缺口 |

**缺口三分法（必须明确归类，禁止混写为"现有研究存在不足"）：**

| 缺口类型 | 含义 | 写法要点 |
|---|---|---|
| 共识区缺口 | 学界有共识，但共识本身存在问题（假设可疑、证据不足、外部有效性存疑） | 先承认共识的贡献，再精准指出其薄弱环节 |
| 争论区缺口 | 学界有真实争论，但争论尚未解决或争论本身存在结构性问题 | 还原争论的真实分歧，指出为何现有争论陷入僵局 |
| 空白区缺口 | 某个问题、视角或数据尚无人系统研究 | 说明为何这个空白重要，不填补会有什么后果 |

每个缺口必须能回答：(1) 从哪篇或哪类具体研究的局限中生长出来？（来源）(2) 不填补会有什么后果？（后果）。来源或后果说不清楚，缺口就是漂浮的。

---

## 第二步附：解释性研究的文献评价方式速查

以下 10 种方式来自王凯、唐世平（2022），用于具体化对既有文献的批评（可选用其中一种或多种）：

**因果理论层面（5种）：**
1. **忽视因果关系**：既有文献忽视了X→Y这对因果关系
2. **缺乏机制讨论**：有关联但未打开因果黑箱
3. **忽视变量组合**：多个变量的特定取值组合才产生因果效应
4. **忽视异质性**：Z对Y的影响存在异质性，X能调节Z的效果
5. **内生性问题**：遗漏变量偏误或反向因果

**理论与实证结合层面（5种）：**
6. **理论与实证脱节**：理论化充分但缺乏检验，或实证有余但缺乏理论化
7. **外部有效性未验证**：因果解释在其它时空情境下未经验证
8. **研究方法单一**：只用了某一类方法，未用互补方法弥补缺陷
9. **案例/样本选择偏误**：以偏概全（便利样本、截断选择、无关案例过多）
10. **预测准确度不足**：模型的预测准确度不够或不能适用于新场景

对于描述性研究，常见评价方向包括：研究对象未被充分描述/描述不够系统精细/案例缺乏代表性/缺乏一手资料/方法单一/概念存在问题。

## 第三步：选择写作策略与切入方式

根据张力性质，两种基本策略可以混用：

**框架整合策略**：先建立一套分析坐标，再把文献放进去。适合需要重新组织既有研究、消解表面分歧、建立新分类的综述。

核心句型示范（参见 Clark, Dolan & Jost 2025）：
- "We identify two dimensions that structure the pathways through which X..."
- "Combining these dimensions yields four types of X..."
- "We synthesize the resurgent literature along these dimensions, illustrating how emerging findings challenge several conventional wisdoms."

**问题拆解策略**：把大问题拆成若干子问题，每个子问题下并列竞争性解释，保留而非消解张力。适合争论本身就是综述重心的情况。

核心句型示范（参见 Rosenfeld & Wallace）：
- "Rival accounts explain this... One view holds... Another perspective argues..."
- "The empirical literature on these questions has grown rapidly, but findings remain mixed."
- "Under certain circumstances, X is effective; under others, it backfires."

处理"混合证据"时：先呈现支持一方的证据，再呈现支持另一方的证据，用条件化语言组织矛盾发现（"the effect depends on," "under what conditions"），不强行解决张力而是厘清张力的结构。

**三种切入方式**（禁止从定义或"学界已有大量研究"开始）：

1. **理论困境切入**：指出主流模型的贡献，然后揭示其盲区
2. **研究演化切入**：通过研究议程的时间序列，引出当前焦点
3. **经验悖论切入**：从一个反直觉的现象或争论出发，引出理论解释的不足

**三种知识贡献框架**（Knopf 2006）：
- 挑战共识：既有共识的假设/证据可能有问题
- 介入争论：帮助裁决竞争性观点或指向综合
- 填补空白：理论空白或经验空白

## 第四步：按需输出骨架

默认确认文献地图后即可进入写作。只在用户明确要求"先看骨架"时，输出以下骨架格式并等待确认：

```text
【文献综述骨架草案】

切入方式：[理论困境 / 研究演化 / 经验悖论]
写作策略：[框架整合 / 问题拆解 / 混合]

第1段：[段落核心主张，1句话]
  └─ 功能：[建立背景 / 确立主流解释的有效范围 / 引出核心张力]
  └─ 主要使用文献：[1-3篇]

第2段：[段落核心主张，1句话]
  └─ 功能：[...]
  └─ 主要使用文献：[...]

缺口落地：既有研究在[具体点]上存在[具体问题]，因此[本研究的介入方式]。
```

---

## 第五步：质量闸门

正文输出前过以下闸门（简化版，原证据闸门已移除——用户提供的文献即为可用文献）：

- **闸门A（引用闸门）**：正文引用必须来自用户提供的文献清单中的条目，不凭空添加未提供文献
- **闸门B（结构闸门）**：每段必须有"主张 + 证据 + 过渡"三要素，避免堆引文
- **闸门C（定位闸门）**：结尾缺口必须可直接映射到用户研究问题

若闸门B或C未通过，回到相应步骤修正。

---

## 第六步：Rubric 自检

输出前按以下三维度自检（每项 0-2 分，总分 6 分，≥4 分通过）：

1. **争议提炼度**：是否形成了可辩论的张力，而非文献罗列
2. **研究定位清晰度**：缺口是否可映射到论文研究问题
3. **引用准确度**：引用是否可追溯到用户提供的文献，无凭空捏造

## 正式写作输出规范

### 文内引用（作者-年份）

- 中文示例：`……这一机制在威权信息环境下更明显（Rosenfeld & Wallace, 2024）。`
- 英文示例：`…under information autocracy (Rosenfeld & Wallace, 2024).`
- 同一论点多文献支持时，按相关性排序，通常不超过3条：`(Huang, 2018; Pan et al., 2022; Blum, 2023)`
- 直接依赖具体论断时给出页码：`(Wedeen, 1999, p. xx)`

### 文末参考文献

- 必须提供完整列表，且**仅包含正文实际引用过的文献**
- 基本字段：作者、年份、题目、来源、卷期页码、DOI/URL（如有）
- 信息不完整时，条目后加`[待补全]`

### 引用密度控制

- 引用服务于论证，不为堆砌；每段 2-4 条高功能引用优先于高密度罗列
- 一段出现十五个引用不如三个引用各自承担明确的论证功能

### 引用真实性约束

- 严禁捏造作者、年份、题目、期刊、卷期页码、DOI
- 信息不完整时如实标注，不伪造细节

| 属性 | 设置 |
|---|---|

| 西文字体与数字 | Times New Roman |
| 字号 | 小四（12pt） |
| 首行缩进 | 2字符（约24pt） |

**执行要点**：

- 推荐文件名：`IR_literature_review_YYYYMMDD.docx`

## 写作中的品质标准

### 节点A：骨架确认后 → 检查"缺口是否有着落"

对每个缺口自查：来源（从哪篇研究的局限中生长出来？）→ 后果（不填补会怎样？）

### 节点B：每段完成后 → 检查"是否出现机械列举"

- 出现"A认为…，B认为…，C认为…"的逐篇总结结构 → 重组为对话形式
- 引用超过5个却没有明确论证功能 → 合并或删除功能不明确的引用
- 段落结尾未揭示张力或缺口 → 补充过渡逻辑

### 节点C：全文完成后 → 检查"综述结尾是否正好是本研究开始的地方"

- 综述最后揭示的缺口是否就是论文要填补的那个缺口？
- 读者读完能不能感受到"确实需要这项新研究"？
- 有无批评了既有文献的某个问题但论文本身不解决的"多余攻击"？

### 五条核心原则

1. **避免"机械列举"**：不要逐篇总结，要把文献组织成对话
2. **缺口要有"着落"**：每个缺口绑定具体研究进展，让读者感到缺口是从文献内部生长出来的。弱写法："现有研究对这一问题关注不足。"强写法："尽管X和Y的研究已经确立了Z机制，但当条件A不成立时，该机制是否仍然适用，尚无系统性检验。"
3. **理论贡献要可见**：批评要有力（sharp）、有据（grounded）、有针对性（targeted）
4. **过渡要有逻辑而非形式**：段落衔接不靠"此外""另外""综上所述"，靠张力传递
5. **文献是论证的材料，不是论证本身**：引用服务于论证功能

### 文献功能分层与篇幅分配（写作前确认）

- **批评性文献（内圈）**：已尝试回答同一问题的文献，详写，从四维中选取最有力角度集中论证
- **支持性文献（外圈）**：概念/方法支撑类文献，一句话说明借鉴了什么、为何适用

### 文献评价常见错误（必须避免）

来自王凯、唐世平（2022）：

1. **没有重点**：对每篇文献平均用力，未找准竞争性文献作为批评靶子。应将主要篇幅用于批评直接竞争性文献
2. **遗漏重要文献**：遗漏了已对本研究问题执行相似设计或提供竞争性解释的文献，或遗漏了对研究设计提出严重质疑的文献
3. **过多无关文献**：评价了与支撑研究正当性/合理性无关的文献，或逐篇数落不足但本研究并未解决所有这些不足

### 文献过多时的筛选规则（Knopf 2006）

- 聚焦被高频引用的权威研究
- 聚焦近期高影响力发表（顶刊、顶尖大学出版社）
- 聚焦与自身研究问题最直接相关的文献

### 文献综述应回答的核心问题（Knopf 2006，自查工具）

- 既有研究提出了什么问题？忽视了什么问题？
- 主要结论是什么？研究实际上论证了什么？
- 共识在哪里？争论在哪里？争论的基础（假设/逻辑/证据/方法）是什么？
- 已有结论的可靠性如何？有无理由质疑某些既有结论？
- 最重要的缺口是什么？需要什么额外研究？

## 子领域识别与注意事项

**安全研究（威慑、冲突、联盟、军备）**
理论争论脉络清晰，流派边界分明。把握"这场争论的真正分歧在哪里"——往往不是表面结论不同，而是假设不同或因果机制不同。

**国际政治经济学（IPE）**
实证研究密度高，方法论分歧显著（量化vs质性，国内政治vs系统层次）。注意"结论的可比性"——不同研究测量的是同一件事吗？

**外交政策分析（FPA）**
跨层次分析是核心挑战。清晰标出每项研究的分析层次（个人/官僚/国内/系统），指出层次间的整合缺口。

**国际制度/国际组织**
大量委托-代理框架，近年微观基础转向明显。注意区分"组织作为行为体"和"个体官僚作为行为体"两类研究的不同假设。

**比较威权/信息政治**
跨学科性强（IR+比较政治）。概念界定是高频争议点（如"宣传"与"审查"的边界）。要对概念争议有所交代，不要直接跳进实证研究。

**建构主义/规范研究**
本体论和认识论立场直接影响研究设计，不能假装分歧不存在。准确还原不同立场的内在逻辑，不只描述结论。
